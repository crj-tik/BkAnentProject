# 房地产行业核心知识

## 交易税费体系
- **契税**：首套90㎡以下1%，90㎡以上1.5%；二套统一3%（部分地区有优惠）
- **个人所得税**：差额的20%或全额的1%；满五唯一免征
- **增值税**：满2年免征（普通住宅）；不满2年按5.6%征收
- **土地增值税**：个人住宅暂免征收

## 限购政策核心规则
- 本地户籍：单身1套，家庭2套（城市差异大）
- 非本地户籍：需连续缴纳社保/个税（通常3-5年），限购1套
- 企业购房：一般禁止，特殊情况需成立3年以上

## 贷款核心参数
- 首套首付比例：30%（普通住宅），非普通住宅可能上调
- 二套首付比例：50%-70%（城市差异）
- 贷款年限：最长30年，房龄+贷款年限≤50年
- 利率：LPR + 基点（首套和二套加点不同）

## 交易流程关键节点
1. 签署定金协议（定金不超过合同价的20%）
2. 网签（存量房买卖合同网上备案）
3. 面签（银行贷款面谈）
4. 资金监管/评估
5. 缴税过户
6. 抵押登记
7. 放款交房

## 常见风险
- 产权瑕疵（查封、抵押、共有产权人未同意）
- 违章建筑（无法过户）
- 学区房政策变动风险
- 租客优先购买权

当用户问题涉及上述任何概念时，请在意图识别中优先考虑以下领域映射：
- 税费计算、贷款评估 → trade（交易可行性分析）
- 合同签署、过户流程 → contract（合同管理）
- 房源搜索、价格对比 → listing + compare（房源搜索+对比）
- 中介费、佣金 → settlement（结算）
